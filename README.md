## 1. Hashator 

Hashator est le projet qui sert a visualiser les collision des algotithme de hashage sur un fichier.png

### Get start


### Fish

```python
git clone https://github.com/BatyaBatkovich/Hashator.git
cd Hashator
python -m venv venv
source venv/bin/activate.fish
pip install -r requirements.txt
python main.py -t custom_hashage -d test.png -f small-rockyou.txt
```
